# Jmem 优化计划

## 功能完善

- [ ] **LLM 记忆提取/摘要**：真正实现 `extractKnowledge()`，使用 LLM 对记忆进行结构化提取和摘要
- [ ] **记忆重要性评分**：为每条记忆添加重要性评分，影响检索排名
- [ ] **记忆衰减机制**：基于时间的记忆衰减，遗忘不常用的记忆
- [ ] **跨会话记忆**：支持在不同会话间共享和检索记忆
- [ ] **用户偏好学习**：从用户交互中学习偏好，优化检索结果

## 技术优化

- [ ] **异步支持**：添加 `MemoryServiceAsync` 接口，支持异步操作
- [ ] **缓存层**：引入 Caffeine 实现查询缓存，减少重复计算
- [ ] **重试机制**：API 调用失败时添加指数退避重试
- [ ] **配置校验**：添加 JSR-303 校验和自定义校验器
- [ ] **健康检查**：添加 Spring Boot Actuator 健康检查端点

## 存储后端

- [ ] **Qdrant 支持完善**：完整实现 Qdrant 向量存储后端
- [ ] **Milvus 后端**：添加 Milvus 向量数据库支持
- [ ] **Redis 后端**：添加 Redis 向量存储支持
- [ ] **多后端一致性**：支持同时使用多个后端

## 架构改进

- [ ] **生命周期管理**：设计记忆何时该删除/合并的策略
- [ ] **增量更新**：修改记忆时只更新变化部分
- [ ] **批量操作优化**：更大批量的批处理支持
- [ ] **事件驱动**：添加记忆变化的回调机制
- [ ] **插件机制**：支持自定义 Embedder、VectorStore 实现

## 运维能力

- [ ] **监控指标**：集成 Micrometer 添加 metrics
- [ ] **链路追踪**：集成 OpenTelemetry 进行分布式追踪
- [ ] **日志规范**：完善日志记录，便于问题排查

## 测试

- [ ] **单元测试**：为核心组件添加单元测试
- [ ] **集成测试**：添加 Spring Boot 集成测试
- [ ] **性能测试**：基准测试检索性能
